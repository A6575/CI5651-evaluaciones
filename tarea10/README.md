# Tarea 10 - Diseño de Algoritmos I

## Sobre el archivo pregunta1.py

El archivo `pregunta1.py` contiene una implementación del algoritmo de Shor para la factorización de números enteros utilizando simulación cuántica clásica.

Este archivo se utiliza para resolver la Pregunta 1 de la tarea, que consiste en factorizar el número `35` en sus factores no triviales.
